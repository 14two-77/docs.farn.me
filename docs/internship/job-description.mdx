# Job Description AI summarizes
- Based on all positions posted in [CEDT Internship Portal](https://cedtintern.cp.eng.chula.ac.th/opening) (2025/11/17, 56 open positions)

## Insights on Needed and Great-to-Have Skills for Each Tech Role

### 1. Frontend Developer (FE)

*   **Needed Skills:**
    *   **Core Languages:** Strong proficiency in HTML, CSS, and JavaScript is fundamental.
    *   **Frameworks/Libraries:** Experience with at least one modern JavaScript framework such as React, Angular, or Vue.js is essential. The provided listings frequently mention React and Angular.
    *   **Responsive Design:** Ability to create websites that work well on various devices (desktop, tablet, mobile).
    *   **Version Control:** Proficiency with Git for source code management.

*   **Great-to-Have Skills:**
    *   **TypeScript:** Knowledge of TypeScript is increasingly sought after for writing more maintainable and scalable code. (gofive, Deverhood HT, JOBTOPGUN, Index Living Mall)
    *   **UI/UX Design Principles:** A good understanding of user experience and user interface design principles to collaborate effectively with designers. Familiarity with tools like Figma is a plus. (B.Grimm Power Public Company Limited, Demeter ICT, Ayodia, Netbay Public Company Limited, Fintech (Thailand) Company Limited, Landometer Co., Ltd.)
    *   **Mobile Development (Flutter):** Experience with cross-platform mobile development using Flutter. (Bluebik, Fintech (Thailand) Company Limited, Orange Cap, xspring advance solutions co., ltd.)
    *   **Testing:** Knowledge of testing frameworks for writing unit and end-to-end tests. (Ayodia, gofive, CHOCO CARD ENTERPRISE Co., Ltd.)

### 2. Backend Developer (BE)

*   **Needed Skills:**
    *   **Programming Languages:** Proficiency in at least one server-side language is required. The listings show a demand for Go (Golang), Python (with frameworks like Django or Flask), Node.js, Java, and C# (.NET Core).
    *   **API Development:** Strong ability to design, build, and maintain RESTful APIs.
    *   **Databases:** Solid understanding of and experience with both SQL (e.g., PostgreSQL, MySQL, SQL Server) and NoSQL (e.g., MongoDB, Redis) databases.
    *   **Version Control:** Proficiency with Git.

*   **Great-to-Have Skills:**
    *   **Cloud Technologies:** Familiarity with cloud platforms like AWS, Google Cloud Platform (GCP), or Microsoft Azure. (B.Grimm Power Public Company Limited, Inteltion, CP Go Soft)
    *   **Containerization (Docker/Kubernetes):** Experience with Docker and container orchestration tools like Kubernetes. (CP Go Soft)
    *   **Microservices Architecture:** Understanding of and experience with designing and building microservices-based applications. (KKP)
    *   **Specific Frameworks:** Deeper knowledge in specific frameworks mentioned, like ASP.NET Web API and Entity Framework. (Ayodia)

### 3. Full Stack Developer

*   **Needed Skills:**
    *   A combination of the "Needed Skills" for both Frontend and Backend developers.
    *   Proficiency in a frontend framework (like React or Angular) and a backend language/framework (like Node.js, Python/Django, or Go).
    *   Strong database knowledge (SQL and/or NoSQL).

*   **Great-to-Have Skills:**
    *   The "Great-to-Have Skills" from both Frontend and Backend roles.
    *   **Mobile Development (Flutter/React Native):** Experience with frameworks like Flutter or React Native for building cross-platform mobile applications. (Bluebik, Fintech (Thailand) Company Limited, Orange Cap, JOBTOPGUN, xspring advance solutions co., ltd.)
    *   **DevOps Knowledge:** Basic understanding of DevOps principles and tools. (CP Go Soft)
    *   **Agile Methodology:** Familiarity with Agile/Scrum development processes. (Ayodia, Bluebik, Index Living Mall, xspring advance solutions co., ltd.)

### 4. DevOps Engineer

*   **Needed Skills:**
    *   **Cloud Infrastructure:** Deep knowledge of at least one major cloud provider (AWS, GCP, Azure).
    *   **Infrastructure as Code (IaC):** Proficiency with tools like Terraform or Ansible.
    *   **CI/CD:** Expertise in setting up and maintaining CI/CD pipelines.
    *   **Containerization:** Strong skills in Docker and Kubernetes.
    *   **Scripting:** Proficiency in scripting languages like Bash or Python.

*   **Great-to-Have Skills:**
    *   **Monitoring and Logging:** Experience with monitoring tools like Prometheus and Grafana. (CP Go Soft)
    *   **Site Reliability Engineering (SRE):** Understanding of SRE principles to ensure system performance and availability. (CP Go Soft)
    *   **Security:** Knowledge of security best practices in a cloud environment. (B.Grimm Power Public Company Limited)

### 5. Data Engineer

*   **Needed Skills:**
    *   **Programming Languages:** Strong skills in Python and SQL are essential.
    *   **ETL/ELT Processes:** Experience in designing and building data pipelines for extracting, transforming, and loading data.
    *   **Big Data Technologies:** Knowledge of frameworks like Apache Spark and Hadoop.
    *   **Data Warehousing:** Experience with data warehousing solutions.
    *   **Cloud Platforms:** Familiarity with data services on AWS, GCP, or Azure.

*   **Great-to-Have Skills:**
    *   **Data Orchestration (Airflow):** Experience with tools like Apache Airflow. (Inteltion, CP Go Soft)
    *   **Specific Cloud Data Tools:** Knowledge of specific tools like AWS DataZone and Redshift. (B.Grimm Power Public Company Limited)
    *   **Data Governance:** Understanding of data management and governance to maintain data integrity. (Inteltion)
    *   **Business Intelligence Tools:** Familiarity with BI tools like Power BI to understand how the data will be consumed. (B.Grimm Power Public Company Limited, Inteltion)

### 6. Data Scientist

*   **Needed Skills:**
    *   **Programming Languages:** High proficiency in Python and its data science libraries (Pandas, NumPy, Scikit-learn).
    *   **Machine Learning:** Strong theoretical and practical knowledge of various machine learning algorithms.
    *   **Statistics:** A solid foundation in statistical concepts and analysis.
    *   **Data Visualization:** Ability to create meaningful data visualizations using libraries or BI tools like Tableau or Power BI.
    *   **SQL:** Proficiency in querying databases.

*   **Great-to-Have Skills:**
    *   **Deep Learning:** Experience with deep learning frameworks like TensorFlow or PyTorch. (CPF, CPF IT CENTER CO., LTD. (Axons))
    *   **Specialized Fields (NLP/Computer Vision):** Knowledge in specific areas like Natural Language Processing (NLP) or Computer Vision (CV), including experience with libraries like OpenCV. (B.Grimm Power Public Company Limited, CPF IT CENTER CO., LTD. (Axons), VISAI)
    *   **Cloud ML Platforms:** Experience with machine learning services on AWS, GCP, or Azure. (Inteltion, CP Go Soft)
    *   **Large Language Models (LLMs):** Understanding and experience working with LLMs. (CPF IT CENTER CO., LTD. (Axons), VISAI)

### 7. AI/Machine Learning Engineer

*   **Needed Skills:**
    *   A combination of skills from Data Scientist and Backend Developer roles.
    *   **Programming:** Strong Python skills.
    *   **ML/DL Frameworks:** Deep experience with frameworks like TensorFlow, PyTorch, or Keras.
    *   **Model Development:** Ability to take machine learning models from research to a functional state.
    *   **Software Engineering Practices:** Solid understanding of software development principles, including version control and testing.

*   **Great-to-Have Skills:**
    *   **Specialized AI Areas (Image Processing/Computer Vision):** Expertise in areas such as Image Processing and Computer Vision, including familiarity with models like YOLO. (CPF)
    *   **RAG and Vector Databases:** Knowledge of Retrieval-Augmented Generation (RAG) and Vector Databases for advanced AI applications. (CPF)
    *   **Prompt Optimization:** Experience with prompt optimization libraries and techniques for working with large language models. (VISAI)
    *   **Data Analytics and Dashboarding:** Ability to create dashboards to visualize model performance and data insights. (CPF)

## Skills Tier List for Applicants

This tier list is a general guideline to help differentiate candidates based on the provided job openings.

### Tier S: High Chance of Being Hired

*   **Technical Proficiency:** Demonstrates deep expertise in the core "Needed Skills" for the specific role. Can articulate complex technical concepts clearly and has hands-on experience from multiple projects.
*   **Problem-Solving:** Can effectively break down complex problems and design and implement robust solutions. Has a strong understanding of algorithms and data structures.
*   **Experience:** Has relevant internship or project experience that directly aligns with the job description. Can showcase a portfolio of work.
*   **Great-to-Have Skills:** Possesses several of the "Great-to-Have Skills," indicating a broader and more advanced skill set (e.g., cloud experience, containerization, specialized AI knowledge).
*   **Soft Skills:** Excellent communication, teamwork, and a proactive, learning-oriented mindset.

### Tier A: Good to Have (Promising Candidate)

*   **Technical Proficiency:** Solid understanding and application of most of the "Needed Skills" but may lack depth in one or two areas.
*   **Problem-Solving:** Good problem-solving abilities but may need some guidance on more complex issues.
*   **Experience:** Has some relevant project or academic experience but may not be a perfect match with the job description.
*   **Great-to-Have Skills:** Has familiarity with one or two of the "Great-to-Have Skills."
*   **Soft Skills:** Good communication and a willingness to learn and collaborate.

### Tier B: Could Be a Fit (Requires Development)

*   **Technical Proficiency:** Basic understanding of the "Needed Skills" but lacks significant hands-on experience. May have more theoretical knowledge than practical application.
*   **Problem-Solving:** Can solve straightforward problems but struggles with more complex or abstract challenges.
*   **Experience:** Limited practical experience; may only have academic projects to show.
*   **Great-to-Have Skills:** Lacks most of the "Great-to-Have Skills."
*   **Soft Skills:** Meets basic expectations for communication and teamwork but may not stand out.

### Tier C: Not Hired (Lacks Essential Skills)

*   **Technical Proficiency:** Lacks a fundamental understanding of the core "Needed Skills" required for the role.
*   **Problem-Solving:** Demonstrates weak analytical and problem-solving skills.
*   **Experience:** No relevant experience or projects.
*   **Soft Skills:** Poor communication skills or a lack of enthusiasm and willingness to learn.
